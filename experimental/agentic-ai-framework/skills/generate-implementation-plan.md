# Skill: Generate Implementation Plan

## Purpose

Convert an AMRIT ticket or concept note into a structured engineering plan.

## Inputs

- Ticket title and description
- Relevant repository/module hints
- Indexed AMRIT docs and code context

## Output

- Affected repositories
- API/UI/data model assumptions
- Implementation steps
- Test plan
- Risks requiring maintainer or domain review

## Safety Note

This skill is an assistant for planning. It should not create implementation tasks without maintainer review, especially for healthcare workflows.
